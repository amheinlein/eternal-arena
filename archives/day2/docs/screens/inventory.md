# Inventory Screen

**Purpose**: View and manage player's items and equipment

## Key Components

- **Equipment Slots**: Visual representation of equipped items
  - Weapon slot
  - Armor slots (head, body, legs, etc.)
  - Accessory slots
- **Inventory Grid**: List of unequipped items
  - Item icons with quantity indicators
  - Tap to view details
- **Item Details Panel**: 
  - Full item stats
  - Equip/Unequip buttons
  - Use button (for consumables)
  - Sell button (if selling is allowed)
- **Filter Tabs**:
  - All Items
  - Equipment
  - Consumables
  - Quest Items
- **Sort Options**: By name, type, stats, etc.

## Flow

1. View inventory
2. Tap item to see details
3. Equip/Unequip items
4. Use consumables
5. Return to main menu

## Navigation

- Equip/Use → Update display, stay on Inventory Screen
- Back → Return to Main Menu
