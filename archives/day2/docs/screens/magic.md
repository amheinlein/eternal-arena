# Magic Screen

**Purpose**: Cast spells for combat and non-combat purposes

## Key Components

- **Spell List**: Available spells organized by category
  - Combat Spells
  - Utility Spells
  - Buff Spells
- **Spell Card**: For each spell
  - Spell name, icon
  - Mana cost (if applicable)
  - Description
  - Cast button
- **Magic Skill Display**: Current Magic skill level
- **Mana/Resource Display**: Current mana or spell resources
- **Spell Details Panel**: Expanded view when spell tapped
  - Full description
  - Effect details
  - Requirements
  - Cast button

## Flow

1. Browse available spells
2. Select spell to cast
3. Confirm target/usage (if needed)
4. Spell effect applied
5. Action consumed
6. Return to main menu or stay on Magic Screen

## Navigation

- Cast Spell → Return to Main Menu (or context-appropriate screen)
- Back → Return to Main Menu
