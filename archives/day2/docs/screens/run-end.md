# Run End/Progression Screen

**Purpose**: Display results when actions are exhausted and show persistent bonuses

## Key Components

- **Run Summary**:
  - Actions used
  - Experience gained
  - Gold earned
  - Items acquired
  - Quests completed
  - Enemies defeated
- **Persistent Bonuses Earned**:
  - List of bonuses unlocked
  - Bonus descriptions
  - Visual indicators for new bonuses
- **Progression Summary**:
  - Total level gained
  - New unlocks achieved
  - Next milestone preview
- **"Start New Run" Button**: Begin next run with persistent bonuses applied

## Flow

1. Actions reach 0
2. Run End Screen appears automatically
3. Show summary and bonuses
4. Player reviews progression
5. Start new run when ready
