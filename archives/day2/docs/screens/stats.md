# Stats/Character Screen

**Purpose**: View detailed character statistics and progression

## Key Components

- **Skill Stats Display**: All seven skills with current levels
  - Attack
  - Strength
  - Defense
  - Magic
  - Ranged
  - Prayer
  - Agility
  - Visual progress bars or numbers
- **Character Summary**:
  - Total level
  - Total experience
  - Gold
  - Runs completed
- **Equipment Stats Summary**: Combined stats from equipped items
- **Persistent Bonuses List**: All active persistent bonuses
- **Achievements/Unlocks** (if applicable):
  - List of unlocked features
  - Progress toward next unlocks
- **Character Visual**: Avatar or character representation

## Flow

1. View comprehensive stats
2. Review progression
3. Check persistent bonuses
4. Return to main menu
