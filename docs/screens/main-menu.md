# Main Menu Screen

**Purpose**: Central navigation hub and action selection screen

## Key Components

- **Action Counter**: Prominent display of remaining actions for current run (e.g., "Actions: 5/10")
- **Action Buttons**: Primary action selection buttons
  - Battle
  - Boss Fight
  - Shop
  - Quest
  - Magic
- **Quick Access Buttons**: Secondary navigation
  - Inventory
  - Stats/Character
  - Settings
- **Run Status Display**: Current run information
  - Run number/ID
  - Current level/experience
  - Gold amount
  - Active quests count (if any)
- **Persistent Stats Preview**: Quick view of key persistent stats
  - Total experience
  - Highest level reached
  - Total runs completed

## Navigation

- Tap Battle → Navigate to Battle List Screen
- Tap Boss Fight → Navigate to Boss Battle List Screen
- Tap Shop → Navigate to Shop Screen
- Tap Quest → Navigate to Quest Screen
- Tap Magic → Navigate to Magic Screen
- Tap Inventory → Navigate to Inventory Screen
- Tap Stats → Navigate to Stats Screen
- Tap Settings → Navigate to Settings Screen

## Special States

- **Actions Exhausted**: When actions reach 0, show "End Run" button prominently
- **New Run Available**: Show "Start New Run" button if no active run
