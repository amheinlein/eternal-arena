# Run Start Screen (Optional)

**Purpose**: Initialize a new run with persistent bonuses applied

## Key Components

- **Persistent Bonuses Applied**: List of bonuses active for this run
- **Starting Stats**: Initial stats for the new run
- **Starting Actions**: Number of actions available
- **Starting Gold**: Initial gold amount
- **"Begin Run" Button**: Start the run

## Flow

1. Player initiates new run
2. Show starting conditions
3. Apply persistent bonuses
4. Begin run
