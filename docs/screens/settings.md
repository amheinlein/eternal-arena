# Settings Screen

**Purpose**: Configure game settings and preferences

## Key Components

- **Game Settings**:
  - Sound volume
  - Music volume
  - Vibration toggle
  - Notifications toggle
- **Display Settings**:
  - Theme (light/dark)
  - Text size
  - Animation speed
- **Gameplay Settings**:
  - Auto-save frequency
  - Confirmation dialogs toggle
  - Tutorial on/off
- **Account/Data**:
  - Save data location
  - Export save data
  - Import save data
  - Reset progress (with confirmation)
- **About**:
  - Version number
  - Credits
  - Links to support/documentation

## Flow

1. Adjust settings
2. Changes saved automatically
3. Return to main menu
